Summary: Latitude vs. X

This project was to analyze how weather changes based on different environmental factors like latitude, wind speed, cloudiness, and humidity.

For this analysis, a data set on over 500 cities was used, which was obtained from OpenWeatherMap API. And, the data was plotted into four different graphs, using Matplotlib.

This website provides the visualizations of the data, analysis on each graph, and the data set that was used for the analysis.
